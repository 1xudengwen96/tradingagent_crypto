## 方法 1：控制台查看（推荐 ⭐⭐⭐⭐⭐）

### 步骤

1. **登录阿里云百炼控制台**
   - 地址：https://bailian.console.aliyun.com
   - 使用你的阿里云账号登录

2. **查看余额**
   - 点击右上角 **用户头像** → **账户管理**
   - 或访问：https://usercenter2.aliyun.com/
   - 查看 **可用余额** 和 **代金券**

3. **查看 Token 用量**
   - 左侧菜单：**费用中心** → **账单详情**
   - 选择 **产品**：百炼（Model Studio）
   - 查看每日/每月 Token 消耗量

4. **查看 API 调用统计**
   - 左侧菜单：**监控与统计** → **API 调用统计**
   - 可按模型、时间范围筛选
   - 查看调用次数、失败率、延迟等

## 方法 3：API 查询（适合自动化 ⭐⭐⭐）

### 阿里云费用中心 API

**接口地址**：
```
POST https://billing.aliyuncs.com
```

**查询账户余额**：
```bash
curl -X POST \
  'https://billing.aliyuncs.com' \
  -H 'Content-Type: application/x-www-form-urlencoded' \
  -d 'Action=DescribeAccountBalance' \
  -d 'Version=2017-12-14' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```

**响应示例**：
```json
{
  "AvailableBalance": 1234.56,
  "CashBalance": 1000.00,
  "CreditBalance": 234.56,
  "Currency": "CNY"
}
```

### 注意事项
- 需要开通阿里云 RAM 访问权限
- 需要创建 AccessKey 和 Secret
- 建议只读权限，避免误操作

## 方法 4：设置余额预警（强烈推荐 ⭐⭐⭐⭐⭐）

### 步骤

1. 登录 **费用中心**
   - 地址：https://usercenter2.aliyun.com/

2. 进入 **预警设置**
   - 左侧菜单：**预警管理** → **余额预警**

3. 配置预警规则
   ```
   预警类型：余额预警
   预警阈值：¥100（或自定义）
   通知方式：短信 + 邮件 + APP 推送
   ```

4. 设置 **消费限额**（可选）
   ```
   每日限额：¥50
   每月限额：¥500
   超出后：停止服务或发送通知
   ```

### 优点
- ✅ 余额不足时自动通知
- ✅ 防止超额消费
- ✅ 避免意外高额账单

## 💡 省钱小技巧

### 1. 设置每日预算
```
每日限额：¥50
超出后：暂停服务
```

### 2. 使用代金券
- 新用户赠送 ¥200-1000 代金券
- 关注阿里云活动获取优惠券

### 3. 选择合适模型
- 测试用 qwen-turbo（最便宜）
- 回测用 qwen-plus（平衡）
- 实盘用 qwen-max（最佳）

### 4. 开启缓存
- 相同请求不重复计费
- 缓存命中率可达 60-80%

## ⚠️ 注意事项

1. **余额不足会自动停止服务**
   - 建议保持 ¥100 以上余额
   - 开启自动充值（可选）

2. **代金券有有效期**
   - 通常 30-90 天
   - 过期作废，优先使用

3. **账单延迟**
   - 实时用量有 5-10 分钟延迟
   - 最终账单以次日为准

4. **退款政策**
   - 充值后不支持退款
   - 建议小额多次充值
